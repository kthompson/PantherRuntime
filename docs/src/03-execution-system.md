# Execution System

## Memory Model

Home values for data:

* arguments
* locals
* fields
* static
* array


## Instructions


### Arguments

ldarg i // by ref arguments
ldarga i // by-value arguments 

### Locals 

ldloc i // by ref arguments
ldloca i // by-value arguments

### Fields

ldflda

### Static

ldsflda

### Array

ldelema
